### Installation
Clone this repo to your computer.

### To Run
Requires [Python3.x](https://www.python.org/downloads/) to run.
```sh
$ cd weather
$ python3 XMLtoJson.py [yourXMLInput.xml]
```
The JSON output file will be generate.
